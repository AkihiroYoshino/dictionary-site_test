# Figma用デザインファイル

このフォルダには「なるほどIT用語辞典」のデザインを検討するためのファイルが含まれています。

## ファイル一覧

| ファイル | 説明 |
|---------|------|
| `design-system.svg` | デザインシステム全体（カラー、タイポグラフィ、UIコンポーネント） |

## Figmaへのインポート方法

### 方法1: ドラッグ＆ドロップ
1. Figmaで新規ファイルを開く
2. `design-system.svg` をキャンバスにドラッグ＆ドロップ
3. 各グループが編集可能な状態でインポートされます

### 方法2: メニューからインポート
1. Figma上部メニュー → **File** → **Place image** (Ctrl+Shift+K)
2. `design-system.svg` を選択
3. 配置位置をクリック

## インポート後の操作

### グループ構造
SVGファイル内の `<g>` 要素は、Figmaでグループとして認識されます：

- `Color-Palette` — カラーパレット
- `Typography` — タイポグラフィ
- `Characters` — キャラクター（先生・生徒・管理人）
- `UI-Components` — UIコンポーネント
- `Page-Layouts` — ページレイアウト
- `Icons` — アイコン

### 色の変更
1. 変更したい要素を選択
2. 右パネルの「Fill」から色を変更

### 画像の差し替え
1. キャラクターやアイコンを選択
2. 右クリック → 「Replace with...」で新しい画像を選択

## 元画像ファイル

images/ フォルダにある以下のSVGファイルも直接インポート可能です：

- `teacher.svg` — 先生キャラクター
- `student.svg` — 生徒キャラクター
- `admin.svg` — 管理人キャラクター
- `logo.svg` — サイトロゴ
