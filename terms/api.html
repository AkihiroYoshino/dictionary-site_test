API

読み：エーピーアイ ／ 正式名称：Application Programming Interface

💡 ひとことで言うと…ソフトウェア同士が会話するための「窓口」のこと！

🗣️ 対話で理解しよう

🧑‍🎓
生徒
先生、「API」ってよく聞くんですけど、一体なんですか？なんか難しそう…
🧑‍🏫
先生
全然難しくないよ！APIは、ソフトウェア同士が会話するための「窓口」だと思えばいいんだ。

たとえば、レストランに行ったとき、お客さんはキッチンに直接入らないよね？注文カウンターを通じて「カレーください」って頼むでしょ。
🧑‍🎓
生徒
うんうん、カウンターで注文して、できたら受け取りますよね。
🧑‍🏫
先生
そう！APIはまさにその「注文カウンター」なんだ。

あるアプリが別のサービスに「天気のデータちょうだい」とお願いすると、APIを通じてデータが返ってくる。
キッチン（サーバー）の中身を知らなくても、カウンター（API）を通じて欲しいものが手に入るんだよ。
🧑‍🎓
生徒
なるほど！じゃあ、天気アプリが天気情報を表示できるのも、裏でAPIを使って天気データをもらってるってことですか？
🧑‍🏫
先生
その通り！🎉 天気アプリは気象庁やOpenWeatherMapなどのAPIに「東京の天気教えて」とリクエストを送って、返ってきたデータを画面に表示してるんだ。

他にも、Googleマップの埋め込み、SNSログイン、決済処理…全部APIの仕組みを使ってるんだよ。
🧑‍🎓
生徒
APIってめちゃくちゃ身近なんですね！全然難しくなかった！

📝 まとめ

  • APIは「Application Programming Interface」の略
  • ソフトウェア同士がデータをやり取りするための窓口・接点
  • レストランの注文カウンターのようなもの
  • 内部の仕組みを知らなくても、決まったルールで頼めばデータがもらえる
  • 天気アプリ、地図の埋め込み、SNSログインなどあらゆる場面で使われている
